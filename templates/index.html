How this works

Encrypt Decrypt
  • 1.Read the original image and extract RGB component
  • 2.Perform DCT over the RBG components of the cover image.
  • 3.Read the watermark image and convert into grayscale image.
  • 4.Apply DCT over Watermark image.
  • 5.Embed DCT coefficients of watermark image over the DCT components (RBG) of cover image. Embedding is performed by alpha blending technique. Equation for alpha blending:
    Final Pixel = alpha * (Cover image pixel) + (1-alpha)*(Water Mark Image Pixel)
  • 6.Perform IDCT of the DCT coefficients obtained after alpha blending to get the watermarked image.
  • 1. Perform DCT over the RBG component of the watermarked/stego image.
  • 2. Perform DCT over the RBG component of the cover image.
  • 3. Extract the original watermark image with the previous value of alpha using the following equation:
    Watermark Image Pixel = ((Watermarked Image Pixel - Cover Image Pixel)*alpha)/(1-alpha);
  • 4. Perform IDCT of the DCT coefficients of each RGB components.
  • 5. Average above obtained values to get the final watermark image
